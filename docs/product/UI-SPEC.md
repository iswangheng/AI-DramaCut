# DramaGen AI UI 设计与交互规范

## 1. 视觉风格 (Visual Identity)
- **主题**: 极简浅色 (Minimalist Light)，背景色 `#F8FAFC` (Slate-50)。
- **核心组件**: 使用微质感卡片，边框 `#E2E8F0`，圆角 `12px`。
- **字体**: 系统默认无衬线体，字重强调层级感（Heading: 600, Body: 400）。

## 2. 页面布局与操作流
### 2.1 仪表盘：素材管理与预处理 (Ingestion Hub)
- **上传区**: 宽屏 Dropzone，支持拖拽多个 4K 素材。
- **状态流**: 每个视频卡片下方有一个分段进度条：
  - [切片中] -> [Gemini 理解中] -> [向量化索引中] -> [就绪]。
- **剧情树组件**: 预处理完成后，点击“查看剧情”，弹出一个交互式的 SVG 树状图，展示 Gemini 拆解的所有故事分支。

### 2.2 模式 A：高光切片量产器 (Highlight Scaler UI)
- **侧边列表**: 左侧纵向排列 AI 识别出的高光候选（缩略图 + 匹配度分数）。
- **中心预览**: 16:9 播放器，下方带有 **“毫秒级微调拨盘”**。
- **微调交互**:
  - 核心按钮组：`[-1s] [-100ms] [当前时间戳输入框] [+100ms] [+1s]`。
  - 键盘快捷键：`A/D` 键控制 ±100ms，`Shift + A/D` 控制 ±1s。
  - 点击“确定并加入渲染队列”，该片段进入右侧的待渲染列表。

### 2.3 模式 B：解说矩阵实验室 (Recap Lab UI)
- **分屏视图**: 
  - **左侧**: Markdown 编辑器风格的解说文案区，支持多版本 Tab 切换。
  - **右侧**: 实时音画匹配流。
- **卡片式匹配**: 每一句解说词对应一个“素材槽位”。
  - 槽位显示当前匹配画面的缩略图和时间戳。
  - 点击“更换”，弹出浮窗展示 Gemini 推荐的 Top 5 候选画面，支持一键替换。

## 3. 渲染监控
- **浮动进度条**: 全局显示当前队列中的视频渲染进度。
- **下载管理**: 提供批量重命名和一键导出至本地目录的功能。

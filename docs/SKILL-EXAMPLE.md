## ✂️ 剪辑规则

### 高能冲突场景

- **时长**: 60-90 秒
- **节奏**: 快节奏剪辑，镜头切换频繁
- **组合方式**: 适合作为视频开场或高潮段落
- **切入**: 从冲突爆发前 5 秒开始（铺垫）
- **切出**: 在冲突达到顶峰后 3 秒结束（余韵）

### 身份揭露场景

- **时长**: 45-60 秒
- **节奏**: 中等节奏，重点在于台词和表情
- **组合方式**: 可单独使用，也可作为高潮段落
- **切入**: 从揭露真相的台词开始
- **切出**: 在人物反应后立即结束（保留悬念）

### 情感高潮场景

- **时长**: 30-45 秒
- **节奏**: 慢节奏渲染，使用慢镜头
- **组合方式**: 适合作为结尾或情感升华点
- **切入**: 从情感爆发点开始
- **切出**: 在情绪释放后慢慢淡出

### 高光 + 钩子组合

- **总时长**: 90-120 秒
- **结构**:
  - 0-60秒: 高光场景（冲突爆发）
  - 60-90秒: 情绪反应（人物震惊）
  - 90-120秒: 钩子结尾（悬念设置）
- **节奏**: 前快后慢，结尾戛然而止
- **组合方式**: 最经典的短视频结构
- **切入**: 从冲突铺垫开始
- **切出**: 在悬念设置点立即黑屏
